Login

Obtener Token

Este método te permite loguear y obtener un token de session

HTTP Request

POST https://abcars.mx/abcars-backend/api/login

Query Parameters
Parameter Description
email string Example: Herrera@abcars.com
Un email válido de un usuario activo de bdc.stregasystem.com
password string Example: mypassword
Una contraseña válida.
gettoken string Example: true
